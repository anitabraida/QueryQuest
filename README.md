# QueryQuest

To start the flask application, run the following commands:

```bash
python3 -m venv venv
```
```bash
. venv/bin/activate
```
```bash
pip install -r requirements.txt
```
```bash
export FLASK_APP=Flask.py      
export FLASK_DEBUG=True
export FLASK_RUN_PORT=8000
```
```bash
flask run
```
